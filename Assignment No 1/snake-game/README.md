# 🐍 Snake Game (Click here to play the Game: https://shakirmahama.github.io/Panaversity/snake-game/)

A Snake Game built with **React** to learn game development concepts through AI-assisted coding.

## Features

* Difficulty selection
* 20 × 20 game board
* Snake rendering
* Head, body, and tail styling
* Responsive UI

## Tech Stack

* React
* JavaScript
* CSS
* Vite

## Run Locally

```bash
npm install
npm run dev
```

## Learning Goal

Understand how games work under the hood by building every feature step by step, including movement, game loops, collision detection, food generation, and state management.
