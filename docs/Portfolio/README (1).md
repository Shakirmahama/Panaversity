# Portfolio Website — Muhammad Shakir

A single-page responsive portfolio built with HTML, CSS, and vanilla JavaScript. Includes a dark/light toggle, sticky nav, and sections for About, Skills, Projects, and Contact.

**Live site:** `https://shakirmahama.github.io/Panaversity/portfolio/`

## Tech Stack
HTML5, CSS3, JavaScript

## Projects Featured
- [Snake Game](https://shakirmahama.github.io/Panaversity/snake-game/) — React game with score tracking and collision detection
- [Expense Tracker](https://shakirmahama.github.io/Panaversity/Expenses-tracker/) — React app for logging and totaling expenses
