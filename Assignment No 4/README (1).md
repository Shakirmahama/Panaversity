# AI-Assisted Data Projects — Summary

**Author:** Muhammad Shakir (shakirmahama786@gmail.com)

---

## `project-1-money-detective/`
Analyzed a bank statement — income, expense categories, duplicates, top spenders, reconciled balance.
**Result:** Rs. 5,401.63 closing balance, reconciled exactly.

## `project-2-whats-my-grade/`
Calculated final course grade from assignments, quizzes, midterm, final, attendance, and participation.
**Result:** 89.30% final grade, verified two ways.

## `project-3-books-dont-match/`
Reconciled farewell dinner payments across inconsistent nicknames/emojis for the same people.
**Result:** Rs. 500 overpayment traced to one person; rest matched exactly.

## `project-4-organize-the-mess/`
Found duplicate photos via perceptual hashing, then organized remaining files by type without touching originals.
**Result:** 3 duplicate groups flagged, 20 files organized, source folder verified untouched.

---

**Approach:** Every project used specific (not vague) prompts, code that actually ran, and a self-verification step to catch errors before reporting final numbers.
