# README: The Grade Calculator Assignment

For this assignment, I asked the AI to calculate my final course grade by writing and running actual code that followed my teacher's exact grading rules (dropped lowest assignment, best 4 of 5 quizzes, the "final beats midterm" swap rule, and the attendance bonus).

1. I gave the AI my raw scores and the exact rules, rather than asking it to just compute a grade from memory — this forced it to apply each rule explicitly instead of guessing.
2. I asked for a **subtotal breakdown per component** (assignments, quizzes, midterm, final, participation, bonus) so I could manually verify each piece myself instead of trusting one final number.
3. I asked it to **self-verify** by recalculating independently. This confirmed the two calculations matched (89.30%) before presenting the final answer.
4. I also asked a follow-up target question ("what final score gets me to 90%") which required solving an equation, not just averaging — the AI had to account for the midterm/final swap rule to answer correctly (90.4 needed).

**Result:** Final grade = 89.30%, verified two independent ways, with every rule shown step-by-step.
